# Manuales de usuario

Esta carpeta contiene la base de manuales por modulo para Smartberries.

Los manuales estan pensados para usuarios finales y describen que permite hacer cada modulo, como navegar por sus secciones principales y que cuidados operativos conviene tener antes de registrar, editar, cerrar o consultar informacion.

## Uso general del sistema

1. Inicie sesion con su usuario y contrasena.
2. Seleccione el modulo que necesita usar.
3. Si el sistema lo solicita, seleccione empresa, planta y temporada.
4. Use el menu lateral para ingresar a registros, listados, reportes o configuraciones.
5. Revise siempre el contexto activo antes de registrar informacion, especialmente empresa, planta y temporada.
6. Para cambiar de modulo, use el boton `Modulos`.
7. Para finalizar, use `Cerrar Sesion`.

## Convenciones de pantalla

- `Registro`: pantalla para crear o editar informacion.
- `Agrupado`: listado resumido de documentos o movimientos.
- `Detallado`: vista con mayor nivel de detalle.
- `Existencia`: consulta de stock disponible, despachado o historial.
- `Apertura Registro`: herramientas administrativas para reabrir registros cerrados.
- `Mantenedores`: tablas base usadas por el sistema.
