# Manual de usuario - Calidad

## Objetivo

El modulo Calidad permite revisar pendientes, configurar parametros de control, registrar controles y consultar reportes por estado.

## Acceso

1. Ingrese al sistema.
2. Seleccione `Calidad`.
3. Confirme empresa, planta y temporada.

## Secciones principales

### Calidad

- Dashboard: vista inicial de control.
- Revision pendientes: registros que requieren evaluacion o seguimiento.

### Configuracion

- Defectos: calidad y condicion.
- Parametros: calibres, presiones y parametros generales.
- Resolucion: reglas de resolucion.
- Inspectores: mantenedor de inspectores.

### Registro

- Recepcion: registro de control de recepcion.
- Proceso: registro de control en proceso.
- Exportacion: registro de control en exportacion.
- Cerrar / PDF recepcion: cierre y generacion de documento.

### Reportes

- Pendientes.
- Abiertos.
- Cerrados.

## Flujo recomendado para control

1. Revise pendientes desde `Revision pendientes` o desde reportes.
2. Ingrese al registro correspondiente.
3. Complete parametros, defectos, resultado e inspector.
4. Guarde el control.
5. Cierre o genere PDF cuando el proceso este completo.

## Cuidados

- Configure defectos, parametros e inspectores antes de operar controles.
- Revise el estado del control antes de cerrarlo.
- Use reportes por estado para seguimiento diario.
