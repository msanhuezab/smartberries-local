# Manual de usuario - Productor

## Objetivo

El modulo Productor permite a usuarios externos o asociados consultar documentos, recepciones, procesos, despachos, existencias y actividad vinculada a su perfil.

## Acceso

1. Ingrese con usuario productor.
2. Seleccione el modulo `Productor` cuando este disponible.
3. Revise la temporada o contexto activo.

## Funciones principales

- Consultar documentos.
- Revisar recepciones de materia prima e industrial.
- Consultar procesos.
- Revisar despachos de materia prima, industrial y producto terminado.
- Consultar existencias.
- Ver perfil, editar datos y cambiar contrasena.
- Revisar actividad del usuario.

## Flujo recomendado

1. Ingrese al informe o documento que necesita revisar.
2. Use filtros disponibles para acotar informacion.
3. Revise el detalle.
4. Descargue o imprima si la pantalla lo permite.

## Cuidados

- El modulo muestra informacion segun permisos y asociaciones del usuario.
- Si falta informacion, confirme que el usuario este asociado al productor correcto.
- Para cambios de datos personales, use las opciones de perfil.
